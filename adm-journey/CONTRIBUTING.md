# Contributing to ADM Journey

## Como Contribuir

### Antes de Começar

1. Faça um fork do repositório
2. Clone o seu fork: `git clone https://github.com/seu-usuario/adm-journey.git`
3. Crie uma branch: `git checkout -b feature/sua-feature`

### Desenvolvendo

1. Instale as dependências: `npm install`
2. Inicie o servidor: `npm run dev`
3. Faça suas mudanças
4. Teste suas mudanças localmente
5. Siga o estilo de código do projeto

### Enviando Mudanças

1. Faça commit das suas mudanças: `git commit -m 'Add feature: descrição'`
2. Faça push para a branch: `git push origin feature/sua-feature`
3. Abra um Pull Request

### Diretrizes

- Use TypeScript
- Siga as convenções de nomenclatura do projeto
- Adicione tipos para novas funções
- Teste suas alterações
- Escreva commits descritivos

## Relatando Bugs

Se encontrou um bug, abra uma issue com:

- Descrição do problema
- Passos para reproduzir
- Comportamento esperado vs. atual
- Screenshots (se aplicável)

## Sugestões

Sinta-se livre para sugerir melhorias. Abra uma issue para discutir!
